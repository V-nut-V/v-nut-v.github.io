Uru-ā-rangi

Realm / description: Connected to the wind/ emphasis on weather

Data types: Air temperature / Pressure / Wind speed / Wind direction

Waipuna-ā-rangi

Realm / description: 'Water that pools in the sky' (Rain, mist, fog, steam, humidity etc)

Data types: Humidity / Rainfall / Temperature / Acoustic rain gauge (measures droplet size)

Waitā

Realm / description: Salt water/ emphasis on sea and kai

Data types: Solunar activity (fish activity) / Significant wave height / Tide height (1st, 2nd, 3rd, 4th tides) / Water temperature rain gauge (measures droplet size)

Waitī

Realm / description: Connected to fresh water/ emphasis on river & kai

Data types: River level in meters / River flow in (m3/sec) / Water temperature / Water temperature / River lifeforms - number/ frequency

Matariki

Realm / description: That which has passed, aspirations for the future

Data types: Number of people in vicinity with mobile phones / Number of received / sent signals inside polling period / Number of people and direction of movement in immediate waka areas / Volume, pitch, rhythm of voices / Solar radiation

Tupu-ā-rangi

Realm / description: 'Tree tops'. Emphasis on kai

Data types: Birdsong (presence, volume, pitch, rythym, number of birds) / Bird movements / Growth and movement of tree branches / Growth and movement of berries

Tupu-ā-nuku

Realm / description: Underground activity. Emphasis on kai

Data types: Seismic activity: Frequency within 50km Hamilton, time, depth, magnitude / Temperature / Ground moisture / Root vibrations
